# FAQ: Rotina

## Manhã caótica - como organizar?
**Pergunta**: Como organizar a manhã com crianças sem enlouquecer?
**Resposta**: A chave é preparar o máximo possível na noite anterior! Deixe mochilas prontas, roupas separadas e lancheira montada. Crie uma rotina visual para as crianças saberem a sequência (acordar → banheiro → roupa → café). Comece acordando 15 minutos antes que elas.
**Missão relacionada**: missao-manha-tranquila
**Gatilho emocional**: Cansaço, estresse matinal, atrasos
**Palavras-chave**: manhã, acordar, escola, atraso, correria, organizar manhã

## Criança que demora para se arrumar
**Pergunta**: Meu filho demora muito para se arrumar de manhã, o que fazer?
**Resposta**: Crianças têm noção diferente de tempo! Use um timer visual (ampulheta ou app) e transforme em desafio: "Será que você consegue colocar a roupa antes do timer acabar?". Evite dar broncas - elas aumentam a resistência.
**Missão relacionada**: missao-manha-tranquila
**Gatilho emocional**: Frustração, atrasos, conflito matinal
**Palavras-chave**: demora, arrumar, vestir, lento, tempo

## Rotina de sono bagunçada
**Pergunta**: Como estabelecer uma rotina de sono para as crianças?
**Resposta**: Consistência é tudo! Defina um horário fixo e crie um ritual relaxante (banho → história → música calma). Reduza telas 1 hora antes. O segredo é manter a rotina mesmo nos fins de semana.
**Missão relacionada**: missao-sono-tranquilo
**Gatilho emocional**: Exaustão, noites mal dormidas
**Palavras-chave**: sono, dormir, noite, acordar, insônia, rotina sono
