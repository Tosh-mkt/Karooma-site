# Privacidade e LGPD

## Quais dados vocês coletam?
**Pergunta**: Quais dados a Karooma coleta?
**Resposta**: Coletamos apenas o essencial: email para newsletter (se você se inscrever), preferências de navegação para melhorar o site, e histórico de conversa com a Karoo para oferecer um atendimento melhor.

## Vocês vendem meus dados?
**Pergunta**: Meus dados são vendidos?
**Resposta**: Jamais! Seus dados são usados apenas para melhorar sua experiência na Karooma. Não vendemos, compartilhamos ou cedemos informações pessoais para terceiros.

## Como excluir meus dados?
**Pergunta**: Como posso pedir para excluir meus dados?
**Resposta**: Você pode solicitar a exclusão dos seus dados a qualquer momento enviando um email para contato@karooma.life. Atenderemos sua solicitação conforme a LGPD.

## O chat é seguro?
**Pergunta**: Minhas conversas com a Karoo são seguras?
**Resposta**: Sim! Suas conversas são usadas apenas para melhorar o atendimento. Não coletamos dados sensíveis como CPF, cartões ou senhas. Nunca pediremos essas informações.

## O que é LGPD?
**Pergunta**: O que significa LGPD?
**Resposta**: LGPD é a Lei Geral de Proteção de Dados do Brasil. Ela garante que você tem controle sobre seus dados pessoais. Na Karooma, seguimos todas as diretrizes para proteger sua privacidade.
